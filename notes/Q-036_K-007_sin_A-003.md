## 2. Rederivación explícita de K-007 sin A-003

### 2.1 Premisas

- **A-001**: existe corte Planck ℓ_P.
- **Walker-Wang framework** (H-003): el espacio-tiempo UV es una red trivalente con sitios separados por ~ℓ_P. Sesión 11 estableció Walker-Wang como el framework dimensional correcto para SCG.
- **Holografía** (Bekenstein-Hawking): S_BH = A/(4ℓ_P²).
- **H-001** (hipótesis): el interior del BH es una cuerda SCG (defecto 1D del lattice WW) plegada en V_BH.

### 2.2 Entropía del BH
```
S_BH = π r_s² / ℓ_P²  = 4π N²       con N ≡ M/M_P = r_s/(2ℓ_P)
```

### 2.3 Entropía máxima de la cuerda

La cuerda SCG es un defecto 1D de la red Walker-Wang. En cada vértice de la red (separados por ~ℓ_P) hay una categoría de fusión con un número finito de objetos; el defecto puede estar en cualquier estado de esa categoría. Por tanto:

```
(# estados por sitio) ≤ |𝒞|           (finito; O(1))
(sitios por unidad de longitud) = 1/ℓ_P
(bits por unidad de longitud) ≤ log₂|𝒞| / ℓ_P ~ 1/ℓ_P  (factor O(1))
```

→ densidad de entropía longitudinal = O(1) bits / ℓ_P.

**Tomamos la saturación:** S_cuerda = L/ℓ_P.

Esto es la aplicación directa de A-001 + WW. No requiere A-003.

### 2.4 Longitud necesaria

S_cuerda = S_BH:
```
L/ℓ_P = 4π N²   →   L = 4π N² ℓ_P = 4π (r_s/2ℓ_P)² · ℓ_P = π r_s²/ℓ_P     (D-003.2')
```

### 2.5 Plegado

La cuerda plegada ocupa V_BH con sección transversal d²:
```
V_BH = (4/3) π r_s³
V_BH = L · d²
d² = V_BH / L = [(4/3) π r_s³] / [π r_s²/ℓ_P] = (4/3) r_s ℓ_P
d = √((4/3) r_s ℓ_P)                                                       (D-003.5')
```

### 2.6 Resultado

```
d = √((4/3) r_s ℓ_P) ~ √(r_s ℓ_P)                    (K-007 preservada)
L = π r_s²/ℓ_P                                        (confirmada)
```

**K-007 se preserva sin usar A-003.** La derivación es estrictamente holográfica + geométrica.

## 3. Verificación de consistencia con D-006 (Casimir de Polyakov)

Ahora que K-027/D-006 da una forma explícita para la energía de modos transversales confinados, hagamos una **verificación de consistencia**: si calculamos E_Casimir de la cuerda interior con las escalas que acabamos de derivar, ¿es coherente con M c²?

### 3.1 Energía Casimir interior

Con L y d de §2.6:

```
E_Casimir = 2π ℏc · L / d²
          = 2π ℏc · (π r_s²/ℓ_P) / ((4/3) r_s ℓ_P)
          = (3π²/2) · ℏc · r_s / ℓ_P²
```

Usando r_s = 2GM/c² y ℓ_P² = ℏG/c³:

```
E_Casimir = (3π²/2) · ℏc · (2GM/c²) / (ℏG/c³) = 3π² M c²
```

**E_Casimir ≈ 30 Mc².** Excede la masa-energía del BH por factor O(30).

### 3.2 Interpretación honesta

Ingenuamente, E_Casimir > Mc² es un problema: la "energía cuántica del interior" excede la masa total observable del BH.

Pero hay tres razones por las que esto NO es una contradicción directa:

**(a) La masa ADM es el invariante a gran distancia.**
La masa M medida desde fuera del BH (la que aparece en la métrica de Schwarzschild) es la masa ADM. Esta incluye todas las correcciones gravitacionales y redshift. La energía "local" dentro del horizonte puede ser muy diferente de M sin contradicción — no es un observable directo para un observador externo.

**(b) Redshift gravitacional infinito en el horizonte.**
Un modo cuántico con energía local E_loc en el interior del BH tiene, para un observador externo, energía E_∞ = E_loc · √(g_{tt}). Cerca del horizonte g_{tt} → 0, así que E_∞ → 0 incluso para E_loc grande. La energía "observable" desde el infinito es mucho menor que la local.

**(c) Balance con gravedad auto-atractiva.**
La cuerda no es libre — está auto-confinada gravitacionalmente. La energía gravitacional propia de la configuración (E_grav) es negativa y grande. La energía ADM es el agregado E_Casimir + E_grav + (masa en reposo), y la cancelación puede ser severa.

### 3.3 Estimación del balance

Energía gravitacional propia (estimación Newtoniana, órdenes de magnitud):
```
|E_grav| ~ G M² / r_s = (G M²) / (2GM/c²) = M c² / 2
```

Así que |E_grav| ~ Mc²/2 y E_Casimir ~ 30 Mc². El Casimir excede la energía gravitacional por factor ~60.

**Esto sí es problemático** a nivel Newtoniano. Señala que el cálculo de E_Casimir usando Polyakov plano y sumándolo al Mc² Newtoniano no es consistente — falta tomar en cuenta la métrica fuertemente curva del interior.

### 3.4 Diagnóstico

El análisis anterior revela una **tensión nueva**, no una refutación de K-007:

> **T-6 (nuevo, sesión 14):** E_Casimir calculado con Polyakov plano + escalas de K-007 excede Mc² por factor ~30. Requiere incorporar la métrica curva interior del BH para reconciliar.

**Estatus:** esperable. D-006 deriva el Casimir en el régimen de Polyakov plano (la cuerda en un fondo asintóticamente plano). Dentro de un BH, la geometría no es plana; el cálculo requiere Polyakov en un fondo de Schwarzschild interior (o su continuación).

**Esta tensión NO invalida K-007** (que es un resultado geométrico-entrópico). Solo indica que el cálculo completo del Casimir en el interior requiere GR fuerte, no solo Polyakov plano.

**Ruta de resolución (futura):** Q-038 abierta — ¿cómo se modifica E_Casimir cuando la cuerda vive en una métrica de Schwarzschild interior con redshift infinito en el horizonte?
