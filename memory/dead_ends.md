# Caminos sin salida

Hipótesis, enfoques y líneas de razonamiento que se intentaron y fallaron. Su propósito: **no repetir el mismo error**.

Cada entrada debe explicar *por qué* falló, para que un razonamiento futuro pueda distinguir entre "este camino está cerrado por X razón" y "este camino se abandonó por falta de tiempo pero podría reabrirse".

Formato:
```
## DE-XXX — Título
- **Fecha de cierre:** YYYY-MM-DD
- **Hipótesis/enfoque:**
- **Por qué falló:** (lógica, experimental, inconsistencia interna, etc.)
- **¿Reabrible?:** sí/no — bajo qué condiciones
```
