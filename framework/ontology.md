# Ontología

> ¿Qué existe según nuestra teoría? ¿Cuáles son las entidades básicas? ¿Qué se deriva de qué?

Este archivo responde a la pregunta: **"¿De qué está hecho el mundo?"** en el marco que estamos construyendo.

## Candidatos para el nivel 0

| Candidato | Idea breve | Problema principal |
|---|---|---|
| Información pura | Bits/qubits como realidad última | ¿Información *sobre qué*? |
| Relaciones | Solo hay relaciones; no hay "relata" | ¿Relaciones entre qué? |
| Procesos | El ser = el devenir (Whitehead) | Difícil de formalizar |
| Estructura matemática | Tegmark MUH: realidad = matemática | No falsable |
| Campo único | Una sola entidad con muchos modos | ¿Qué selecciona los modos? |
| Red causal | Nodos + enlaces causales | ¿Qué son los nodos? |
